開発体制: 本人(Claude活用)7割 + 開発部3割(ユーザー管理・契約まわり)

---

## 1. 背景、課題

### 市場背景
- 国内のハウスクリーニング事業者数は全国**1万〜1.5万社**程度(2024〜2025年推計)。市場規模は約8,000億円〜1兆円規模で、年平均4.5%前後で成長中。
- 参入障壁が低く、フランチャイズ大手(ダスキン、おそうじ本舗、おそうじ革命等)以外は地域密着の中小・零細事業者、個人事業主が大半を占める。
- 高齢化・共働き世帯の増加・単身世帯の増加により、今後も需要拡大が見込まれる成長市場。

### 課題
- 利用経験者への調査で、業者選びの際に最も重視される項目は「**料金・見積もりの明瞭さ**」(47.0%)であり、信頼性・スタッフ対応(合わせて42.0%)を上回る最重要項目。
- 一方で、業界全体として見積もり内容が不透明・業者間で見積額の差が大きいという不信感が根強く、成約率・リピート率を下げる要因になっている。
- 零細事業者は見積書・提案書を作るリソース(人手・時間・デザインスキル)を持たないケースが多く、口頭・簡易メモでの見積もり対応にとどまっている事業者も多い。
- 弊社が過去に開発した「RENO」(リフォーム提案AI: AIヒアリング→ビフォーアフター画像生成→概算見積もり→PDF提案書自動生成)が、同種の課題を持つリフォーム業界で価値を実証済み。この型をハウスクリーニング業界に転用することで、開発コストを抑えつつ新市場を狙える。

## 2. ソリューション

ハウスクリーニング事業者(主に零細〜中小規模)向けに、以下の一連の流れをAIエージェントとして提供する。

1. **AIチャットヒアリング**
 見込み客がLINEまたはWeb上でAIと会話しながら、「掃除したい範囲」「部屋の広さ」「気になる汚れ・要望」「希望日時」を入力。人に直接話しにくい要望(散らかり具合への引け目など)もAI相手だと伝えやすい。

2. **ビフォーアフター画像生成**
 見込み客が撮影した部屋写真をもとに、AIが「片付いた後の同じ部屋」のイメージ画像を生成。契約前に完成イメージを見せることで、成約率・安心感を高める。

3. **概算見積もりシミュレーター**
 部屋の広さ・作業内容・写真から推定した汚れ/物量の程度をもとに、概算費用をその場で自動算出。

4. **PDF提案書の自動生成**
 見積もり内訳・作業内容・ビフォーアフター画像を1枚のPDFにまとめ、事業者が見込み客にすぐ送付できる状態にする。

RENOの既存エンジン(AIヒアリング・画像生成・PDF生成の基盤部分)を転用することで、ゼロから開発するより開発期間・コストを抑えられる。
